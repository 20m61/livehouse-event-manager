Livehouse Event Manager
=======================

Requires at least: 6.0  
Tested up to: 6.x  
Requires PHP: 7.4  

== Description ==
ライブハウスやイベントスペース向けの使いやすいWordPressイベント管理プラグイン。月間カレンダー表示、リスト表示、カスタム投稿タイプ、ジャンル分類、AJAXナビゲーション機能を備えています。

== Installation ==
1. プラグインディレクトリ（wp-livehouse-event-manager）を `wp-content/plugins/` にアップロード  
2. WordPress 管理画面から有効化
